SetWebhook

Для регистрации вебхука используйте данный метод GET https://apps.bein.tech/uyi/api/v1/{apikey}/setWebhook?url={YOUR_CALLBACK_URL} YOUR_CALLBACK_URL это ваш URL куда будут отправлятся данные о событиях
Пример : 
https://apps.bein.tech/uyi/api/v1/biclients/setWebhook?url=http://api.bi.group/smarthome/events
в формате Данные будут поступать в формате
{
  • 1:
    {
    • title: "ПЛК",
    • devices:

      • {
        • name: "Движение",

      ]
    }
}

Service Alerts

Сервис получения уведомлений о событиях КСК POST https://apps.bein.tech/uyi/api/v1/{apikey}/alert Внутри тела передать текст сообщения который надо вывести на экраны домофонов ( формат может быть ввиде html )

Master переключатель Отключить Все

Сервис получения уведомлений о событиях КСК POST https://apps.bein.tech/uyi/api/v1/{apikey}/master/{on|off} Переключатель режима "Выключить все"
Пример : 
https://apps.bein.tech/uyi/api/v1/biclients/master/on

NoopWebhook

Для дебага вебхука GET https://apps.bein.tech/uyi/api/v1/noopWebhook

Set

Для дебага устройств GET https://apps.bein.tech/uyi/api/v1/{apikey}/set/{slaveId}/{registry}/{value} Где slaveId - это номер modbus slaveа, registry - номер регистра ( номер сенсора или реле ) , value - значение регистра
